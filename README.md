# Range Slider

Custom Range slider 

```jsx
     <RangeSlider
        min={0}
        max={100}
        step={1}
        value={value}
        onChange={(e) => {
          setValue(e.target.value);
        }}
      />
```

# Run & Build
```
 npm install
 npm run start
 npm run build
```
